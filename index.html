Website Rental PS

Menggunakan

GitHub

Website Rental PS merupakan website yang bertujuan memudahkan pelanggan dalam melakukan penyewaan Playstation(PS), pelanggan dapat melakukan penyewaan PS kapan saya dan dimana saja sehingga penyewa tidak perlu datang ketempat untuk melakukan penyewaan PS, yang dimana PS akan di antar ke alamat tujuan sesuai dengan yang terdaftar di dalam website. Wesbite ini menampilkan tampilan Front-End untuk user dan Back-End untuk Admin. Yang dimana user dapat melakukan daftar,login dan menyewa ps, sedangkan admin dapat melakukan CRUD terhadap kategori dan console. Admin pun dapat memantau stok dan laporan terhadap consolenya. Wesite ini dibuat menggunakan Framework PHP Codeigniter3 dan Database Mysql.
